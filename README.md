## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS, Framer Motion (for smooth 3D-like physics and transitions)
- **Backend & Database:** Node.js, Express.js, MongoDB
- **Integrations:** EmailJS (automated messaging service), Netlify (hosting & continuous deployment)

---

## ✨ Features

- **Interactive UI:** Smooth-scrolling navigation with dynamic viewport tracking (`IntersectionObserver`).
- **Engaging Visuals:** Modern glassmorphism design layouts integrated with custom animations.
- **Fully Operational Mail Panel:** Visitors can drop a line directly via the portfolio. Submissions automatically route to my personal inbox with a seamless automated customer confirmation responder.
- **Responsive Architecture:** Pixel-perfect optimization across mobile, tablet, and desktop viewports.
